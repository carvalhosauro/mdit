# mdit — markdown, live in your terminal

Edit markdown and see it **rendered as you type**. No preview pane.

## Why

- One buffer to *edit* and *read* — no split view
- [[wikilinks]] with autocomplete, follow, and back
- Zen mode for distraction-free reading

> The block under your cursor stays raw. Everything else renders.

`go install github.com/carvalhosauro/mdit/cmd/mdit@latest`
